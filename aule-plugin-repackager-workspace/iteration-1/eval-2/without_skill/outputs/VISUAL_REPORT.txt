     Result: Located 4 plugin source directories

Step 2: MAPPING
  └─ Locate corresponding .zip files
     Result: Found .zip for each plugin in plugin directory

Step 3: EXTRACT TIMESTAMPS
  ├─ Source: find src-dir -type f -exec stat -c %Y {} \; | sort -rn | head -1
  │  (Gets most recent file modification time in entire directory tree)
  └─ Zip:    stat -c %Y *.zip
     (Gets .zip file modification time)

Step 4: COMPARISON
  └─ time_diff = src_timestamp - zip_timestamp
     If time_diff > 0: Source is newer → OUT_OF_SYNC
     If time_diff ≤ 0: Zip is current → IN_SYNC

Step 5: REPORTING
  └─ Generated text reports and JSON data for analysis
