# Olytic Plugins

Plugin ecosystem for Olytic Solutions. Built for Claude Code.

## Plugins

### The One Ring
Governance plugin — brand voice, company strategy, HR policies, and security policies. Installed for all employees at managed scope.

### Olytic Content Strategist
Content creation and strategy plugin — website content, GitHub integration, competitive research, GA4 analytics. Requires The One Ring.

## Installation

```bash
# Governance (all employees — managed scope)
claude plugin install the-one-ring --scope managed

# Content team
claude plugin install olytic-content-strategist
```
